# Voice Assistant 🎤

This is a Python-based Voice Assistant that can listen to voice commands and respond with speech.

## Features
- Responds to greetings
- Tells current time
- Searches information using Wikipedia
- Opens YouTube

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Wikipedia API

## How to Run
1. Install required libraries:
   pip install SpeechRecognition pyttsx3 pyaudio wikipedia

2. Run the program:
   python voice_assistant.py
